# FINAL QA & Rule Enrichment V10

## 목적
최신 inference_result.xlsx / inference_result_v9_check_after_extra.csv에서 발견된 기타보험 오탐, 간병/간병인 니즈 불일치, 치아/턱관절 담보 누락, 공공 의료보험/OOS 혼입을 종합 점검해 룰 사전과 Resolver를 보강했다.

## 주요 개선
- coverage_focus가 선택되어도 `가격/보험료/비교/추천/후기/필요성/장단점`이 있으면 customer_need_type을 상업/탐색 의도에 맞게 보정
- `간병인 보험 가격` → 보험료확인, `간병인 보험 비교` → 상품비교
- `간병인 지원 제도`, `간병비 지원`, `간호 간병 서비스`, `의료 보험 조회` 등 공공/비민영 검색어 OOS 처리
- `가족 간병`, `부모님 간병`, `상주/입주/호스피스 간병인` 등 간병 문맥 exact rule 추가
- `실란트`, `지르코니아`, `스플린트`, `턱관절 치료/장애` 치아보험 보강
- coverage typo threshold를 0.90으로 상향해 `리치 간병 → 치매간병비`, `족간병 → 재가간병비` 같은 과잉 오타 매칭을 차단

## 삭제/정리
이전 패치별 감사 문서는 제거하고, 최종 설계 계약과 본 V10 검수 리포트만 유지했다.
